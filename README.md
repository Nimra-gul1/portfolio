# Portfolio Site

This repository contains a static portfolio website for Nimra Gul.

## GitHub Pages

The site is configured to deploy automatically to GitHub Pages using GitHub Actions.

### What to do in GitHub

1. Push these changes to the `main` branch.
2. Open your repository on GitHub.
3. Go to Settings → Pages.
4. Under "Build and deployment", choose "GitHub Actions" as the source.
5. The workflow in [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml) will publish the site automatically.

Once the workflow completes, the site will be available at:

https://<your-username>.github.io/<your-repo-name>/
